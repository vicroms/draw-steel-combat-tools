A Foundry VTT module for the Draw Steel system that provides a suite of tools to streamline or fully automate the combat mechanics of Draw Steel, including Forced Movement, Conditions (general and some class-specific), Teleportation, and NPC Death. It also includes miscellaneous utilities such as labels assigned by Groups in combat, and automation for the I'm No Threat Harlequin Shadow maneuver, previously distributed as standalone macros but bundled here for convenience and easier updates. Each feature can be individually toggled in the settings, so you can use as much or as little of the module as fits your table.
